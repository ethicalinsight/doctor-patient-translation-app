# Medical Chat Application (Nao Medical Internship Assignment)

A functional React-based chat interface prototype designed for communication between a **Doctor** and a **Patient**. This project demonstrates core frontend development skills, browser API integration, and the ability to leverage AI tools to build and deploy a modern web application under a tight deadline.

## üåü Key Features

* **Dual-Role Simulation:** Toggle between "Doctor" and "Patient" views to simulate a two-way conversation with dynamic message alignment.
* **Audio Recording & Playback:** Utilizes the **Web MediaRecorder API** to record voice notes and generate playable audio blobs directly in the chat.
* **Real-time Search:** A search bar that filters the message history dynamically using React state management.
* **Simulated Translation:** Demonstration of translation logic to handle multi-language support in a medical context.
* **Auto-Scroll:** UI automatically scrolls to the most recent message for a better user experience.
